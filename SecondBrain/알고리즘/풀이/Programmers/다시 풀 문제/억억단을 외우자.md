## 정답 코드

```java
import java.util.*;

class Solution {
    public int[] solution(int e, int[] starts) {
        // 1. 약수의 개수 구하기
        int[] divisorCount = new int[e + 1];
        for (int i = 1; i <= e; i++) {
            for (int j = i; j <= e; j += i) {
                divisorCount[j]++;
            }
        }

        // 2. 뒤에서부터 해당 지점에서의 최적의 답(최대 등장 횟수를 가진 가장 작은 수) 계산
        int[] bestValue = new int[e + 1];
        int currentMaxCount = 0;
        int currentBestNum = 0;

        for (int i = e; i >= 1; i--) {
            // 등장 횟수가 더 많거나, 같더라도 수가 더 작으면 갱신
            // (역순으로 진행하므로 '작거나 같을 때' 갱신하면 자연스럽게 가장 작은 수가 유지됨)
            if (divisorCount[i] >= currentMaxCount) {
                currentMaxCount = divisorCount[i];
                currentBestNum = i;
            }
            bestValue[i] = currentBestNum;
        }

        // 3. 결과 매핑
        int[] answer = new int[starts.length];
        for (int i = 0; i < starts.length; i++) {
            answer[i] = bestValue[starts[i]];
        }
        
        return answer;
    }
}
```
### 틀린점 
- 배수 카운팅이라는 것을 몰랐다. 약수의 개수를 구할 수 있는 쉬운 방법
	- 원하는 것이 약수의 개수임은 알았지만 이를 에라토스테네스의 체를 활용해 풀려고 한 점이 패착
