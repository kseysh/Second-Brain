## 나의 틀린 풀이
```java
import java.io.*;
import java.util.*;

class Main {

    static int n, m;
    static List<int[]>[] graph; // end, time
    static int[] reachTimes; // 각 노드 지역까지 가는데 필요한 최소 시간

    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));

        StringTokenizer st = new StringTokenizer(br.readLine());
        n = Integer.parseInt(st.nextToken());
        m = Integer.parseInt(st.nextToken());

        graph = new ArrayList[n];
        for (int i = 0; i < n; i++) {
            graph[i] = new ArrayList<>();
        }
        reachTimes = new int[n];

        for(int i = 1; i <= m; i++){
            st = new StringTokenizer(br.readLine());
            int start = Integer.parseInt(st.nextToken()) - 1;
            int end = Integer.parseInt(st.nextToken()) - 1;

            graph[start].add(new int[]{end, i});
            graph[end].add(new int[]{start, i});
        }

        Arrays.fill(reachTimes, Integer.MAX_VALUE);
        Queue<int[]> q = new ArrayDeque<>(); // node, before time
        reachTimes[0] = 0;
        q.add(new int[]{0, 0});
        while(!q.isEmpty()){
            int[] cur = q.poll();
            int node = cur[0];
            int beforeTime = cur[1];

            for(int[] edge : graph[node]){
                int next = edge[0];
                int nextTime = edge[1];
                int totalTime;

                if(nextTime < beforeTime){
                    totalTime = reachTimes[node] - beforeTime + m + nextTime;
                }else{
                    totalTime = reachTimes[node] - beforeTime + nextTime;
                }

                // 기존 값 보다 현재 계산한 값이 더 작다면 add
                if(reachTimes[next] > totalTime){
                    reachTimes[next] = totalTime;
                    q.add(new int[]{next, nextTime});
                }
            }
        }
        System.out.println(reachTimes[n - 1]);
    }
}
```
## 정석 풀이
```java
import java.io.*;
import java.util.*;

public class Main {
    static class Node implements Comparable<Node> {
        int id;
        long time;

        public Node(int id, long time) {
            this.id = id;
            this.time = time;
        }

        @Override
        public int compareTo(Node o) {
            return Long.compare(this.time, o.time);
        }
    }

    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        StringTokenizer st = new StringTokenizer(br.readLine());

        int n = Integer.parseInt(st.nextToken());
        int m = Integer.parseInt(st.nextToken());

        // 인접 리스트 구성: 해당 지역에서 갈 수 있는 {연결지역, 신호순서(0~M-1)}
        List<int[]>[] adj = new ArrayList[n + 1];
        for (int i = 1; i <= n; i++) adj[i] = new ArrayList<>();

        for (int i = 0; i < m; i++) {
            st = new StringTokenizer(br.readLine());
            int u = Integer.parseInt(st.nextToken());
            int v = Integer.parseInt(st.nextToken());
            adj[u].add(new int[]{v, i});
            adj[v].add(new int[]{u, i});
        }

        long[] dist = new long[n + 1];
        Arrays.fill(dist, Long.MAX_VALUE);
        
        PriorityQueue<Node> pq = new PriorityQueue<>();
        
        dist[1] = 0;
        pq.add(new Node(1, 0));

        while (!pq.isEmpty()) {
            Node curr = pq.poll();

            if (curr.time > dist[curr.id]) continue;
            if (curr.id == n) break; // 목적지 도착 시 최단거리 보장

            for (int[] edge : adj[curr.id]) {
                int nextNode = edge[0];
                int signalOrder = edge[1]; // 신호가 들어오는 시점 (0 ~ M-1)

                long currentTime = curr.time;
                long remain = currentTime % m;
                long waitTime;

                // 대기 시간 계산 핵심 로직
                if (signalOrder >= remain) {
                    waitTime = signalOrder - remain;
                } else {
                    waitTime = (m - remain) + signalOrder;
                }

                long nextTime = currentTime + waitTime + 1; // 대기 + 이동(1분)

                if (dist[nextNode] > nextTime) {
                    dist[nextNode] = nextTime;
                    pq.add(new Node(nextNode, nextTime));
                }
            }
        }

        System.out.println(dist[n]);
    }
}
```

### 틀린 이유
1. `Arrays.fill(graph, new ArrayList<>())`를 하면 모든 graph에 같은 ArrayList가 들어간다. 이를 주의하자.
2. 어떤 노드에 빨리 도착하는 것이 최종 목적지에 빨리 도착하는 것을 보장하지 않는 문제인데 이를 BFS로 풀었기 때문 (다익스트라를 잘못 적용함)
	1. 다익스트라의 대전제: 가중치는 변하지 않아야 한다.
	2. 하지만, 이 문제의 경우 내가 도착한 시간에 따라 가중치가 변함
	3. PQ를 활용한 다익스트라의 경우 모든 노드에 대해 절대 시간이 가장 빠른 순서대로 탐색을 확장하기 때문에 풀이가 가능함
