## hashset struct 정의
```cpp
namespace std {
    template <>
    struct hash<pair<int, int>> {
        size_t operator()(const pair<int, int> &v) const {
            return v.first * 31 + v.second; // 소수(31)를 사용한 해시 결합
        }
    };
}
```

## 이진 탐색
```cpp
binary_search(v.begin(), v.end(), point);

lower_bound(v.begin(), v.end(), point);
```
lower_bound 함수는 내가 찾고자 하는 그 대상이 저장된 인덱스 중 가장 작은 인덱스를 반환한다.
upper_bound 함수는 내가 찾고자 하는 그 대상이 저장된 인덱스 중 가장 큰 인덱스+1을 반환한다.
