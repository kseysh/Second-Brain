# 코드 정리: 테스트 코드 정리
> 테스트 코드도 코드이기 때문에 유지보수 대상이다. 즉 테스트 메서드에서 발생하는 중복을 알맞게 제거하거나 의미가 잘 드러나게 코드를 수정할 필요가 있다.

### 하지만!!
![[Pasted image 20231016105359.png]]
테스트코드는 실제 서비스에서 돌아가는 코드가 아니기 때문에 테스트 코드 관리가 용이해지는 것이 첫 번째 목표여야하지 아닐까 싶다.
