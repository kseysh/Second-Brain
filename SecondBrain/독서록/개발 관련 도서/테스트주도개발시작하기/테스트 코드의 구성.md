실행 결과를 확인하는 쉬운 방법 => 리턴 값을 사용하여 확인하는 것

> 상황-실행-결과 확인 구조에 너무 집착하지 말자. 이 구조가 테스트 코드를 작성하는데 도움이 되는 것은 맞지만 꼭 모든 테스트 메서드를 이 구조로 만들어야 하는 것은 아니다. 테스트 코드를 보고 테스트 내용을 이해할 수 있으면 된다.
