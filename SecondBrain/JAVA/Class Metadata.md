JVM의 Method 영역에 저장되는 정보
## 클래스 메타데이터에 포함되는 정보
- 클래스 이름 (FQN)
- 패키지 정보
- 접근 제어자 (`public`, `abstract`, `final`)
- 상속 관계 (`extends`, `implements`)
- 필드 목록
    - 이름, 타입, 접근자
- 메서드 목록
    - 이름, 파라미터 타입, 반환 타입
- 생성자 정보
- 애노테이션 정보 (`@Entity`, `@Repository` 등)
- 제네릭 정보 (소거 전 타입 일부)
- 바이트코드 정보

## 사용되는 것
리플렉션에서 런타임에 클래스 구조를 조사하기 위해 사용
