## Tree 용어
- Degree - children의 수
- External node(leaf): degree가 0인 node
- Internal node: degree가 양수인 node
- Ancestor: root부터 x의 부모까지의 unique한 path
- Descendant: 조상의 inverse
- Subtree: x를 root로 한 x의 후손들
- Depth: 부모의 depth +1
	- root는 depth 0
- Level: 같은  depth를 가진 모든 노드
- Height: leaves의 depth의 최댓값
