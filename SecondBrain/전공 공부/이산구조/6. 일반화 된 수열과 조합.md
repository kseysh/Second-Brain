# 중복 조합
`nHr` => `n+r-1Cr`
서로 다른 n개의 집합에서 중복을 허용하여 r개를 뽑는 행위

## 파스칼 항등식
<sub>n+1</sub>C<sub>k</sub> = <sub>n</sub>C<sub>k-1</sub> + <sub>n</sub>C<sub>k</sub>
