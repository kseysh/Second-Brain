# 분할 정복 알고리즘 공식
시간 복잡도 점화식이다.
`f(n) = af(n/b) + g(n)`
n : sorting 해야하는 수
a개의 sub problem으로 나눔
각각의 subproblem의 수 : b
g(n) : extra operation의 수 (ex) binary search에서 트리 쪼개는 연산, merge sort에서 합병하는 연산)
## 이진 탐색 공식
`f(n) = f(n/2) + 2`
## 합병 정렬 공식
`f(n) = 2f(n/2)+n`
## 빠른 정수 곱셈
`f(n)=3f(n/2)+Cn`

f(n) = af(n/b)+cn<sup>d</sup>
