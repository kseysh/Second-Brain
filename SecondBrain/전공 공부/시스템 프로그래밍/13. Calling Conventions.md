## Stack Frame
메모리 내의 스택(일종의 메모리 영역)에서 함수 호출과 관련된 정보를 저장하는 데이터 구조
stack Frame은 
매개변수
지역 변수
리턴 포인터를 저장할 수 있다.
임시저장공간으로도 활용가능하다

callee returns before caller does 
callee : 함수 안의 함수
caller : 함수를 부른 함수
#### 관리
공간은 함수 안으로 들어갔을 때 할당된다.
call 함수에 의해 push를 수행한다.
return될 때 Deallocated를 수행한다.
ret 함수에 의해 pop을 수행한다.
