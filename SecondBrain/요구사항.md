- 사용자들이 평균 찌른 사람 명수
```sql
SELECT AVG(poked_count) AS avg_poked_users
FROM (
    SELECT poker_id, COUNT(DISTINCT poked_id) AS poked_count
    FROM app_dev.poke_history
    GROUP BY poker_id
) AS poker_stats;
// 8.8235294117647059
```
- 가장 많이 찌른 사람 명수

SELECT MAX(poked_count) AS max_poked_users
