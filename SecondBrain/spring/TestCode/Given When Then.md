1. 모든 테스트 문장은 Given When Then으로 나눠서 작성할 수 있어야 한다.
    
2. Given/When/Then
    
    - Given
        - 테스트를 위해 주어진 상태
        - 테스트 대상에게 주어진 조건
        - 테스트가 동작하기 위해 주어진 환경
    - When
        - 테스트 대상에게 가해진 어떠한 상태
        - 테스트 대상에게 주어진 어떠한 조건
        - 테스트 대상의 상태를 변경시키기 위한 환경
    - Then
        - 앞선 과정의 결과

**즉, 어떤 상태에서 출발(given)하여 어떤 상태이 변화를 가했을 때(when) 기대하는 어떠한 상태가 되어야 한다.(then)**
