# DML (Data Manipulation)종류
- SELECT
- INSERT
- UPDATE
- DELETE
- **MERGE**
- 저장(commit) 혹은 취소(rollback) 반드시 필요
# **DCL (Data controll)종류**
- GRANT
- REVOKE
# DDL (Data Definition)종류
- CREATE
- ALTER
- DROP
- **TRUNCATE**
- **RENAME**
# TCL(Transaction Controll) 종류
-  COMMIT
- ROLLBACK
