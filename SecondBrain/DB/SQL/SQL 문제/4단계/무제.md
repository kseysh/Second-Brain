```sql
SELECT SUB.FLAVOR
FROM 
    (SELECT 
        (SUB.TOTAL_SUM + FH.TOTAL_ORDER) AS TOTAL_ORDER,
        FH.FLAVOR,
        ROW_NUMBER() OVER (ORDER BY SUB.TOTAL_SUM + FH.TOTAL_ORDER DESC) AS NUM
    FROM (
            SELECT 
                FLAVOR,
                SUM(TOTAL_ORDER) AS TOTAL_SUM
            FROM JULY
            GROUP BY FLAVOR
    ) AS SUB
    RIGHT JOIN FIRST_HALF FH ON FH.FLAVOR = SUB.FLAVOR
    GROUP BY FH.FLAVOR
) AS SUB
WHERE SUB.NUM <= 3;
```


## 최적화 풀이
```sql
SELECT FH.FLAVOR
FROM FIRST_HALF FH
LEFT JOIN (
    SELECT FLAVOR, SUM(TOTAL_ORDER) AS TOTAL_SUM
    FROM JULY
    GROUP BY FLAVOR
) J ON FH.FLAVOR = J.FLAVOR
ORDER BY (FH.TOTAL_ORDER + IFNULL(J.TOTAL_SUM, 0)) DESC
LIMIT 3;
```
limit 사용법을 알아두자.
