```sql
WITH RECURSIVE TIME_H AS (
    SELECT 0 AS HOUR            -- 초기값 설정
    UNION ALL
    SELECT HOUR + 1 FROM TIME_H -- 반복 수행
    WHERE HOUR < 23             -- 정지 조건
)

SELECT 
    T.HOUR, 
    COUNT(A.ANIMAL_ID) AS COUNT
FROM 
    TIME_H T
LEFT JOIN ANIMAL_OUTS A ON T.HOUR = HOUR(A.DATETIME)
GROUP BY T.HOUR
ORDER BY T.HOUR;
```

## 틀린 내용
with 절을 활용해 recursive 만들기
join절 쓰는 방법
