```sql
SELECT 
    C.CAR_ID, 
    C.CAR_TYPE, 
    TRUNCATE(C.DAILY_FEE * 30 * (1 - P.DISCOUNT_RATE / 100), 0) AS FEE
FROM CAR_RENTAL_COMPANY_CAR C
JOIN CAR_RENTAL_COMPANY_DISCOUNT_PLAN P 
  ON C.CAR_TYPE = P.CAR_TYPE AND P.DURATION_TYPE = '30일 이상'
WHERE C.CAR_TYPE IN ('세단', 'SUV')
  AND C.CAR_ID NOT IN (
      SELECT CAR_ID
      FROM CAR_RENTAL_COMPANY_RENTAL_HISTORY
      WHERE START_DATE <= '2022-11-30' AND END_DATE >= '2022-11-01'
  )
HAVING FEE >= 500000 AND FEE < 2000000
ORDER BY FEE DESC, CAR_TYPE ASC, CAR_ID DESC;
```

## 배운점
1. **FROM / JOIN**: 데이터를 가져오고 테이블을 합칩니다.
2. **WHERE**: 행을 하나씩 검사하며 필터링합니다. **(← 여기서 FEE를 찾으려고 함)**
3. **GROUP BY**: 데이터를 그룹화합니다.
4. **HAVING**: 그룹화된 결과에 조건을 겁니다.
5. **SELECT**: 어떤 컬럼을 보여줄지 결정하고 계산을 수행합니다. **(← FEE라는 별칭이 여기서 탄생함)**
6. **ORDER BY**: 결과를 정렬합니다.
MYSQL은 SELECT에서 만든 별칭을 HAVING 단계에서 참조할 수 있다.
