## 나의 풀이
```SQL
SELECT SUB.CATEGORY, SUB.PRICE AS MAX_PRICE, SUB.PRODUCT_NAME
FROM ( 
    SELECT 
        F.*,
        ROW_NUMBER() OVER (PARTITION BY F.CATEGORY ORDER BY F.PRICE DESC) AS R
    FROM     
        FOOD_PRODUCT AS F
    WHERE F.CATEGORY IN ('과자', '국', '김치', '식용유')
) AS SUB
WHERE 
    SUB.R = 1
ORDER BY 
    SUB.PRICE DESC;
    
```

- `ROW_NUMBER`: 하나만 나오는 것을 보장하므로 이걸 신경써야할 때는 신경쓰자
