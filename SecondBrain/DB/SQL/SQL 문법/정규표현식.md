## 주요 정규 표현식 패턴
- ^ : 특정 문자로 시작하는 패턴
- $ : 특정 문자로 끝나는 패턴
- .(dot) : 임의의 문자
- \[abc] : 특정집합(abc)에 속하는 문자열
- {n} : 해당 패턴이 n회 반복
- {n, m} : 해당 패턴이 n회 이상, m회 이하 등장
- ^[abc] : a, b, c 중 하나로 시작하는 문자 
- ^[abc].* : a, b, c 중 하나로 시작하는 문자열
- \[^abc] : 특정집합(abc)에 속하지 않는 문자열
- \[a-z] : a부터 z까지
- * : 패턴이 0번 이상 반복
- + : 패턴이 1번 이상 반복
- | : or을 의미
