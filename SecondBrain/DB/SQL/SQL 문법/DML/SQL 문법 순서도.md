데이터베이스는 **"데이터를 어디서 가져올지(FROM)"** 결정하는 것부터 시작해서 **"어떤 모양으로 보여줄지(SELECT)"** 순으로 진행합니다.

1. **FROM (+ JOIN):** 데이터를 조회할 테이블을 지정하고 조인하여 전체 데이터 셋을 구성합니다.
2. **WHERE:** 개별 행(Row)에 조건을 걸어 필터링합니다. (집계 전의 필터링)
3. **GROUP BY:** 특정 컬럼을 기준으로 데이터를 그룹화합니다. (데이터가 '덩어리'가 됨)  
4. **HAVING:** 그룹화된 결과(집계된 값)에 대해 조건을 걸어 필터링합니다.
5. **SELECT:** 화면에 출력할 컬럼을 선택하고, 별칭(Alias)을 부여하며 계산(산술 연산 등)을 수행합니다. **(스칼라 서브쿼리는 보통 이 단계에서 실행됩니다.)**
6. **DISTINCT:** 중복된 결과 행을 제거합니다.
7. **ORDER BY:** 최종 결과의 정렬 순서를 결정합니다.
8. **LIMIT / OFFSET:** 최종 출력할 행의 개수를 제한합니다.
